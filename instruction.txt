22. *Now you have face to a problem that you can't access infolder pages from a direct link ex: https://i-computer-frontend-bay.vercel.app/products. for resolve this problem create a file name vercel.json in root folder in front-end.
    *Copy and paste this code into file, {
        "rewrites": [
            {
            "source": "/(.*)",
            "destination": "/index.html"
            }
        ]
        }

    *This means user have to go home page and index.html first if he comes any other sources.
    *After deploy this to github you can see in vercel queued trying to ready the hosting , this means ci/cd pipeline.
