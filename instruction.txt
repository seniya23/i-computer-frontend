    *Paste this code in terminal "npm install tailwindcss @tailwindcss/vite"
    *To import the package paste this line in vite.config.js import tailwindcss from '@tailwindcss/vite'
    *And paste this code tailwindcss() inside this pluggin section "export default defineConfig({
                                                                        plugins: [react(), tailwindcss()],
                                                                        })"

    *use this command to import tailwindcss "@import "tailwindcss";" into css file,so i import tailwindcss inside index.css file
